BOX ファイルの作成
====================================================================================================
作成した仮想マシンを元に Box ファイルを作成します。作成した Box ファイルを ``vagrant box add`` コマンドで
登録して使用できます。作成した Box ファイルを配布することで、同じ環境を提供できます。
