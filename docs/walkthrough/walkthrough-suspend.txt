仮想マシンをサスペンド
----------------------------------------------------------------------------------------------------
稼働中の仮想マシンをサスペンドします。

.. code-block:: none

   vagrant suspend

.. code-block:: none
   :emphasize-lines: 1

   PS C:\vagrant\centos7> vagrant suspend
   ==> default: Saving VM state and suspending execution...
   PS C:\vagrant\centos7>

Oracle VM VirtualBox マネージャーで確認すると、仮想マシンがサスペンドしていることがわかります。
